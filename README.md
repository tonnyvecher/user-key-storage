Доброе пожаловать, дорогой друг!

Представляю твоему вниманию мой дипломный проект, который называется "Разработка системы хранения ключевой информации для идентификации пользователя"


Проект дорабатывается, в связи с различными обстоятельствами


На данный момент тут представлено следующее:

docker-контейнер, в котором находятся такие сервисы, как:

1. PostgreSQL (для хранения данных пользователя, ролей и т.д.),
2. HashiCorp Vault (для хранения ключевой ифнормации, а именно мастер-ключей для шифровки/расшифровки чувствительные данные для идентификации пользователя и других секретных операций над данными(в последствии)),
3. Java-wrapper (обёртка на ЯП Java, которая взаимодействует с ключами из Vault для шифрования данных или операций и backend-стороной для отдачи данных, которые потребовали расшифрование),
4. Keycloack (выступает как внешний провайдер во внешйней ИАА модели),
5. backend-сторона (для создания api для связи с БД и Java-обёрткой (бэк не контактирует с Vault хранилищем и его ключами)),
6. frontend-часть (Vue SPA клиент с авторизацией через Keycloack; чтение и создание заметок, завязанных на уровни доступа; чтение данных других пользователей по uid и в отдельной странице списка пользователей; редактирование своего профиля(телефон/дата рождения))


Инструкция к запуску:
1. развернуть весь проект локально
2. скачать и установить Docker Desktop
3. запустить билд докер-контейнера командой в PowerShell из корня репозитория:

        docker-compose up --build

5. скачать и установить Node.js
6. запустить фронт-клиент в отдельном окне PowerShell командой:

        npm run dev

8. по адресу Keycloack http://localhost:8081/ - заходим в админ-realm и создаем пользователей во вкладке users
9. тестировать и тыкать!
