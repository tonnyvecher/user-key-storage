Представляю твоему вниманию мой дипломный проект, который называется "Разработка системы хранения ключевой информации для идентификации пользователя"

Проект дорабатывается, в связи с различными обстоятельствами

На данный момент тут представлено следующее:
    - docker-контейнер, в котором находятся такие сервисы, как:
        - PostgreSQL (для хранения данных пользователя, ролей и т.д.),
        - HashiCorp Vault (для хранения ключевой ифнормации, а именно мастер-ключей для шифровки/расшифровки чувствительные данные для идентификации пользователя и других секретных операций над данными(в последствии)),
        - Java-wrapper (обёртка на ЯП Java, которая взаимодействует с ключами из Vault для шифрования данных или операций и backend-стороной для отдачи данных, которые потребовали расшифрование), 
        - Keycloack (выступает как внешний провайдер во внешйней ИАА модели), 
        - backend-сторона (для создания api для связи с БД и Java-обёрткой (бэк не контактирует с Vault хранилищем и его ключами))
